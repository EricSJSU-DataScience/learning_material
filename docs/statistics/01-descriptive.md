# Descriptive Statistics

## Central Tendency

| Measure | Description | Python |
|---------|-------------|--------|
| Mean | Average of all values | `np.mean(data)` |
| Median | Middle value | `np.median(data)` |
| Mode | Most frequent value | `stats.mode(data)` |

## Spread

| Measure | Description | Python |
|---------|-------------|--------|
| Variance | Average squared deviation | `np.var(data)` |
| Std Dev | Square root of variance | `np.std(data)` |
| Range | Max - Min | `np.max(data) - np.min(data)` |
