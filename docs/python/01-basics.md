# Python Basics

## Variables

```python
x = 10
name = "Alice"
is_active = True
```

## Data Types

| Type | Example |
|------|---------|
| int | `42` |
| float | `3.14` |
| str | `"hello"` |
| list | `[1, 2, 3]` |
| dict | `{"key": "value"}` |

## Conditionals

```python
if x > 0:
    print("positive")
elif x == 0:
    print("zero")
else:
    print("negative")
```
