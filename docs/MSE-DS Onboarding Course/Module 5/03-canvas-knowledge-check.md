# Canvas Knowledge Check

This quiz is designed to test your basic knowledge of Canvas, Penn Engineering Online's learning management system. If you haven't done so already, we encourage you to read through the [Canvas overview](https://online.seas.upenn.edu/student-knowledge-base/canvas-overview/) in our Student Knowledge Base. Be sure to pay particularly close attention to the additional knowledge base Canvas articles at the bottom of the page.
