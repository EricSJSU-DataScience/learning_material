# Canvas Overview

Canvas is the Learning Management system in which all of your courses will be located. You can access Canvas as a Penn student using your Penn Key and password at this link: [https://canvas.upenn.edu/](https://canvas.upenn.edu/)

Canvas is where you can view your class dashboard, including all lectures, videos, assignments, class recordings and grades.

It is important you understand how to use this essential platform. To learn more about navigating Canvas, [see our Canvas info page on the Student Knowledge Base](https://online.seas.upenn.edu/student-knowledge-base/canvas-overview/).
