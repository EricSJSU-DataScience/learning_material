# MSE-DS Module 3: Course Registration

This module provides an overview of the course selection process, an introduction to the course registration system, and important registration deadlines to remember.

---

## Learning Objectives

Upon completion of this module, you will be able to:

- Successfully utilize Path@Penn as a registration system to register for courses
- Identify important registration deadlines

---

## What You'll Learn

**Course Selection Process**  
Understand how to evaluate and select courses that align with your academic goals and timeline. Learn how to choose from required, core, and elective courses in the MSE-DS curriculum.

**Path@Penn Registration System**  
Discover how to use Path@Penn, Penn's course registration platform, to enroll in courses. Learn navigation, registration procedures, and troubleshooting common issues.

**Registration Deadlines**  
Familiarize yourself with critical registration dates and deadlines. Understand when courses open for registration, add/drop deadlines, and late registration policies.

---

## Key Points

- Path@Penn is Penn Engineering Online's official course registration system
- Understanding course selection criteria helps you build a strong plan of study
- Registration deadlines are firm and vary by semester
- Early registration ensures you secure spaces in desired courses
- Support is available if you encounter registration issues
