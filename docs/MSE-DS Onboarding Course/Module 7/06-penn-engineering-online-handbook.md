# Penn Engineering Online Handbook

Students should refer to the [Penn Engineering Online Handbook](https://onlinelearning.seas.upenn.edu/MCIT-Handbook/) for policies that apply to online Penn Engineering students.

Please make sure you are familiar with the policies in this handbook such as the Engineering Code of Ethics, Grievance Procedures, and requirements for graduation.
