# Academic Integrity Policy at Penn Engineering Online

Posting or soliciting solutions to coursework or exams is a violation of Academic Integrity. Students shouldn't post information about the guidelines, format, rubrics, and/or actual grades associated with exams or assignments. **Penn Engineering Online has a zero-tolerance policy for posting assignments or solutions on outside websites (e.g., Chegg, Coursehero, Bartleby, public GitHub repos), social media, or chat services (e.g., WeChat, Discourse, via text, etc.)**. It is also unacceptable to reference solutions posted on these sites for your coursework. Penn Engineering Online staff monitor these sites regularly, and students who are found to be responsible will be referred to the Center for Community Standards and Accountability (CSA). If found guilty, the student may fail the class and/or be suspended from Penn.

---

## GitHub and Code Management Systems

*Note: If you use GitHub (or similar cloud-based code management system) to set up a remote code repository, you are required to keep that repository private unless otherwise specified by your instructors or course staff.*

If you are unsure whether your use of GitHub constitutes a violation of the Code of Academic Integrity, **it is your responsibility to consult with a TA or instructor to clarify any ambiguities**.
