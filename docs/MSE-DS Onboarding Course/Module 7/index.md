# Module 7: University Policies and Academic Integrity

In this module you will learn more about University of Pennsylvania and Penn Engineering Online specific policies. This module will primarily highlight academic integrity at Penn. This module's goal is to give you a clear understanding of your responsibilities as a Penn student. You will also learn about Honorlock and practice using this platform as it will be used in your courses.

---

## Learning Objectives

- Recall the location of all University Policies, including the Code of Student Conduct and the Code of Academic Integrity
- Recall the location of all policies that apply to online Penn Engineering students
- Understand your responsibilities as a Penn student
- Know where to go for help to clarify any ambiguities on University Policies or policy violations
- Define each of the academic integrity categories
- Describe how to use Honorlock and how it will be used in your course
