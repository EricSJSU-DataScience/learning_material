# MSE-DS Module 1 Description

In this module, you will be introduced to the faculty and staff that have helped grow the MSE-DS Online program as well as Penn's history. Slack will be highlighted in this module and how the platform will be used in your program.

---

## Learning Objectives

Upon completion of this module, you will be able to:

- Identify MSE-DS Online faculty and Programs staff
- Describe how to use Slack and how it will be used in your program

---

## Key Points

- Meet the faculty and staff shaping the MSE-DS Online program
- Learn about Penn's history and institutional background
- Understand Slack as a primary communication tool in your program
- Familiarize yourself with how Slack will support your learning experience
