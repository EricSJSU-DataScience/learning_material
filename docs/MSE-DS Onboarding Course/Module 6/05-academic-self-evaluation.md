# Academic Self-Evaluation and Plan for Success

The following exercise will help you evaluate your academic strengths and needs in an online learning environment. For each of the topics, note whether it is a strength or an area for development. If it is an area for improvement, determine if it is a major or minor priority.

**This exercise is ungraded and purely for your own edification and it will NOT be graded.**
