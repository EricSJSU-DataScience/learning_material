# Data Analysis Commands

Linux command-line tools for processing and analyzing data.

- [grep](grep.md) — search text by pattern
- [sed](sed.md) — find and replace in text
- [awk](awk.md) — process structured text / CSV
- [cut](cut.md) — extract columns
- [sort & uniq](sort-uniq.md) — sort and deduplicate
- [wc](wc.md) — count lines, words, characters
- [Pipes & Redirection](pipes.md) — combine commands
