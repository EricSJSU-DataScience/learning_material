# What is Linux

Linux is an open-source operating system kernel, first released by Linus Torvalds in 1991. It powers servers, cloud infrastructure, Android devices, and most data science environments.

## Key Concepts

**Kernel** — the core of the OS; manages hardware, memory, and processes.

**Distribution (distro)** — a packaged version of Linux bundled with tools and a package manager. Common ones:

| Distro | Common Use |
|--------|------------|
| Ubuntu | Beginner-friendly, widely used in data science |
| Debian | Stable, foundation for Ubuntu |
| CentOS / RHEL | Enterprise servers |
| Arch | Minimal, for advanced users |

## Why Linux for Data Science?

- Most cloud servers (AWS, GCP, Azure) run Linux
- Python, conda, Docker, and ML tools work natively on Linux
- Command-line tools for data processing (grep, awk, sed) are built in
- Free and open source
