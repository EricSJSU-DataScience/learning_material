# cd — Change Directory

## Usage

```bash
cd [directory]
```

## Examples

```bash
cd /home/eric/projects    # go to absolute path
cd projects               # go to relative path
cd ..                     # go up one level
cd ../..                  # go up two levels
cd ~                      # go to home directory
cd -                      # go back to previous directory
```

## Tips

```bash
cd ~/Downloads            # ~ always expands to your home dir
cd "My Folder"            # use quotes if name has spaces
```
