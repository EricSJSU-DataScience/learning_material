# touch — Create File

Creates an empty file, or updates the timestamp of an existing one.

## Usage

```bash
touch filename
```

## Examples

```bash
touch notes.txt               # create empty file
touch file1.txt file2.txt     # create multiple files
touch data/output.csv         # create in a subdirectory
```
