# System Operations

Daily commands for navigating and managing files.

- [pwd](pwd.md) — print current directory
- [cd](cd.md) — change directory
- [ls](ls.md) — list files
- [mkdir](mkdir.md) — create directory
- [touch](touch.md) — create file
- [cp](cp.md) — copy
- [mv](mv.md) — move / rename
- [rm](rm.md) — remove
- [cat](cat.md) — display file contents
- [find](find.md) — search for files
