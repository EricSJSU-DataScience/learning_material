# Advanced NumPy Topics

Deep dive into powerful features for optimized numerical computing.

## Contents

- [Broadcasting](broadcasting.md) - Efficient operations on arrays of different shapes
- [Advanced Indexing](advanced-indexing.md) - Complex selection and manipulation
- [Data Types](data-types.md) - Type system and conversions
- [Structured Arrays](structured-arrays.md) - Heterogeneous data structures
- [Universal Functions (ufuncs)](ufuncs.md) - Element-wise operations and vectorization

## Topics Overview

**Broadcasting**: Apply operations to arrays of different shapes intelligently using NumPy's broadcasting rules.

**Advanced Indexing**: Use boolean arrays, integer arrays, and complex indexing patterns for powerful data selection.

**Data Types**: Deep dive into NumPy's type system, casting rules, and type promotion.

**Structured Arrays**: Work with record-like data using field names and mixed data types.

**Universal Functions**: Understand ufuncs and how they vectorize operations for performance.
