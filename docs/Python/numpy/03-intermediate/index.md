# Intermediate NumPy Topics

Beyond the basics - manipulating arrays, understanding memory, and working with complex operations.

## Contents

- [Shape Manipulation](shape-manipulation.md) - Reshape, flatten, and reorganize arrays
- [Copies and Views](copies-views.md) - Understanding NumPy's memory model
- [Stacking and Splitting](stacking-splitting.md) - Combine and separate arrays

## Topics Covered

**Shape Manipulation**: Reshape, flatten, transpose, expand dimensions, and reorganize array structure without changing data.

**Copies vs Views**: Understand when NumPy creates references to data (views) versus independent copies, and how this affects memory and performance.

**Stacking & Splitting**: Combine multiple arrays using vstack/hstack, and separate arrays using hsplit/vsplit for data preparation tasks.

## Key Concepts

**Shape Manipulation**: Change array structure without changing data
**Memory Management**: Understand when NumPy copies vs references data
**Array Combination**: Stack, concatenate, and split arrays efficiently
