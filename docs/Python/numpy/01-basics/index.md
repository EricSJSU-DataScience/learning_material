## Key Concepts

**What is an array?**
An array is a grid-like structure for storing and retrieving data. NumPy's `ndarray` (N-dimensional array) generalizes this concept to any number of dimensions.

**Why arrays over lists?**
- Faster performance on numerical operations
- Less memory consumption
- Vectorized operations (no explicit loops needed)
- Consistent data types
